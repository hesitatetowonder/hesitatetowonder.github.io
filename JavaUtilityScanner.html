What happens when you include "import java.util.Scanner;"

When you include "import java.util.Scanner in your code it imports the Scanner class that resides in the utility folder that is included in the main java folder into your code so that the functionality of that class is available to you.

The Scanner class is a class in java.util which allows the user to read values of various types (primarily keyboard input and text files) and store them in variables that can be manipulated in the program.
